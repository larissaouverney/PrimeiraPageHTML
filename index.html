Client

Voltar

Client é o receptor de toda mensagem enviada através da Web. Ele é quem solicita os dados para o servidor e exibe para seu usuário toda informação coletada através daquela solicitação.

Por exemplo, um usuário utiliza seu smartphone para fazer uma pesquisa no Google, o client (que nesse contexto seria o smartphone), envia aquela solicitação para a web e recebe a resposta obtida, podendo entregá-la ao seu usuário.

As linguagens client-side são linguagens onde apenas o seu NAVEGADOR vai entender.

Quem vai processar essa linguagem não é o servidor, mas o seu browser.

Server

Voltar

Enquanto o Client é o responsável por enviar a mensagem do usuário e receber o retorno, o Server é onde fica retida toda informação.

Utilizando o mesmo exemplo citado acima, o Server é o responsável por armazenar o conteúdo que o Client está buscando

As inguagens server-side são linguagens que o SERVIDOR entende.

Isso quer dizer que vai escrever um código onde o servidor vai processá-lo e então vai mandar para o seu navegador a resposta.

HTML

Voltar

Diferente do que se acredita, HTML não é uma linguagem de programação, mas sim uma linguagem de marcação de texto, por isto seu nome HyperText Markup Language ou Linguagem de Marcação de HiperTexto.

Criado por Tim Berners-Lee, o HTML tem como sua principal função estruturar uma página web e organizar toda informação contida ali. Sem ele, a página não conseguiria exibir imagens, textos e quaisquer outros conteúdos.

Ao visitar uma página simples na web, você pode perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa estrutura é estabelecida através do HTML.

{...} Portanto, podemos considerar o HTML o “esqueleto” da sua página.

Tags

Voltar

As tags são delimitadores dentro do html, elas são responsáveis pelas marcações dentro do código através das instruções que possuem. Além disso, elas também são responsáveis por criar os elementos nas páginas web, garantindo organização.

Alguns exemplos de tags bastantes utilizadas são:

Body

Esta é responsável por delimitar todo o corpo do documento.

OL

Ela cria uma lista ordenada, por exemplo:

  1. Item 1
  2. Item 2

UL

Diferente da tag ol, a ul é responsável por criar uma lista não ordenada:

  • Item
  • Item

LI

Esta tag é utilizada junto das tags acima, UL e OL, pois ela tem como objetivo listar os objetos que ficarão dentro das listas, ordenadas e não ordenadas.

H1 ao H6

As tags h1, h2, h3, e que seguem até o h6, são reponsáveis por estabelecer o tamanho da fonte de um título. Sendo H1 a maior e diminuindo até a H6. Exemplo:

Este texto é H1

Este texto é H2

Este texto é H3

Este texto é H4

Este texto é H5
Este texto é H6

Strong

Esta tag tem como objetivo deixar um texto completo ou palavras dentro deste em negrito.

Br

Esta tag é utilizada para pular uma linha após um texto.

Font

Font é uma tag antiga que não é mais utilizada no HTML5, font era utilizada para alterar as propriedades de um texto, como seu tamanho, tipo e cor da fonte.

Blockquote

Blockquote tem como objetivo introduzir uma citação, por exemplo:

Tem uma doença que às vezes dá nas raposas que as faz deixar de agir de maneira normal e atacar estranhos. A guerra é uma doença humana parecida.

Além das tags apresentadas acima, existem muitas outras que desempenham diversas funções e têm também muita importância dentro do html.
